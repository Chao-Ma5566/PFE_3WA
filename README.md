# PFE_3WA

Pour lancer le projet:

    Pas besoin d'entrer dans le dossier, lancez les commandes suivants dans 3 terminals:

    ==========================================================================================
    npm run dev
    cd front && npm start
    cd front && npx tailwindcss -i ./src/index.css -o ./public/output.css --watch
    ==========================================================================================
